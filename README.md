# Digital Clock | Timer | Stopwatch

A responsive Digital Clock Web App built using HTML, CSS, and JavaScript, featuring a real-time clock, a customizable timer, and a functional stopwatch â€” all in one place.

# Features
# Digital Clock

Displays current hours, minutes, and seconds in real-time

Updates automatically using setInterval()

Clean and minimal interface

# Timer

Set custom countdown time

Start, pause, and reset timer

Displays time dynamically with smooth transitions

# Stopwatch

Start, stop, and reset stopwatch

Records laps

Precise millisecond tracking

Fully interactive and responsive

# Tech Stack

HTML5

CSS3

JavaScript (ES6)
